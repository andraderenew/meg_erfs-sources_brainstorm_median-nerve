# Data Sources & Disk Notes

## Dataset
- **Name:** Brainstorm Median-nerve tutorial dataset
- **Official source:** https://neuroimage.usc.edu/brainstorm/Tutorials
- **Subset used in this repository:** 1 subject
- **Condition reported here:** Left median-nerve trials

## Local organization
- The dataset was imported into a local Brainstorm protocol/database on the analysis workstation.
- Raw MEG files, anatomy, Brainstorm database files, and large derivatives are **not committed** to this repository.
- Only lightweight documentation and exported portfolio outputs (PNG figures and small text/metadata files) are tracked in Git.

## Disk notes
- Exact local raw-data and Brainstorm-database sizes were not recorded for this portfolio entry.
- Repository size is intentionally kept small by excluding raw neuroimaging data and large Brainstorm derivatives.

## Reproducibility note
The public tutorial source and the processing description in `README.md` provide the provenance needed to recreate the analysis workflow without redistributing the raw dataset through GitHub.
